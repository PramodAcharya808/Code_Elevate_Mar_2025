Basic Syntax

Selectors
Element selector
h1{
  color: #FBBC04;
}

Class selector
.myclass{
  color: #FBBC04;
}

ID selector
#myId{
  color: #FBBC04;
}

Universal selector
*{
  font-size: 18px;
}
